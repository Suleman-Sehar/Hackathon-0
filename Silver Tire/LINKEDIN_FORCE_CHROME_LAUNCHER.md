## 💻 **WHAT THE BATCH FILE DOES**

```batch
@echo off
REM Finds Chrome installation
if exist "C:\Program Files\Google\Chrome\Application\chrome.exe" (
    start "" "C:\Program Files\Google\Chrome\Application\chrome.exe" "https://www.linkedin.com/login"
)
```

**Why it works:**
- ✅ Directly calls Chrome executable
- ✅ Passes LinkedIn URL as parameter
- ✅ Windows opens specified program
- ✅ No ambiguity about which browser
