# SKILL: Social Summary Generator

**Version:** 0.3 Gold Tier - Phase 2
**Owner:** Suleman AI Employee v0.3

---

## Objective

After posting to social media, generate a short performance summary and save to domain-specific Social_Reports folder.

---

## MCP Integration

**Called automatically after social_mcp.py execution:**

```python
# In orchestrator or skill execution
def generate_summary(platform, message, result):
    summary = {
        "platform": platform,
        "posted_at": datetime.now().isoformat(),
        "message_preview": message[:100] + "..." if len(message) > 100 else message,
        "status": result.get("status"),
        "expected_reach": estimate_reach(platform, message)
    }
    save_summary(summary)
```

---

## Steps

### 1. Capture Post Data

From MCP response:
```json
{
    "status": "success",
    "platform": "twitter",
    "url": "https://twitter.com/status/..."
}
```

### 2. Generate Summary Content

**Template:**
```markdown
# Social Post Summary

**Platform:** [Facebook/Instagram/Twitter]
**Posted:** [YYYY-MM-DD HH:MM:SS]
**Domain:** [Business/Personal]

## Content Preview
"[First 100 characters of message]..."

## Performance
- Status: ✅ Posted successfully / ❌ Failed
- Session: Saved to /[platform]_session/
- Audit Log: `/Logs/audit_YYYY-MM-DD.json`

## Expected Reach
[High/Medium/Low] - Based on:
- Time of posting
- Content type
- Historical engagement

## Next Steps
- Monitor engagement for 24h
- Track metrics in dashboard
```

### 3. Estimate Expected Reach

| Factor | High | Medium | Low |
|--------|------|--------|-----|
| Time | 9-11 AM, 7-9 PM | 11 AM-5 PM | 12-6 AM |
| Content | Image + text | Text only | Link only |
| Hashtags | 3-5 relevant | 1-2 | None |

### 4. Save Summary

**Location:**
- Business: `/Business/Social_Reports/[platform]_YYYY-MM-DD_HH-MM-SS.md`
- Personal: `/Personal/Social_Reports/[platform]_YYYY-MM-DD_HH-MM-SS.md`

### 5. Update Dashboard

Add to `/Dashboard.md`:
```markdown
## Recent Social Activity

| Platform | Time | Status |
|----------|------|--------|
| Twitter | 15:30 | ✅ Posted |
| Instagram | 14:00 | ✅ Posted |
```

## Aggregation (Weekly)

```python
def weekly_summary():
    """Aggregate all posts from the week."""
    reports_dir = Path("/Business/Social_Reports/")
    weekly_posts = []
    
    for report in reports_dir.glob("*.md"):
        if report_is_this_week(report):
            weekly_posts.append(parse_report(report))
    
    return {
        "total_posts": len(weekly_posts),
        "by_platform": count_by_platform(weekly_posts),
        "best_performing": find_best_performing(weekly_posts)
    }
```
