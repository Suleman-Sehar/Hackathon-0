## 🔍 **TROUBLESHOOTING**

### QR Code Not Appearing?

**Wait longer** - takes 5-15 seconds to load

**Check internet** - make sure you're connected

**Refresh page** - click browser refresh button

---

### Phone Can't Scan QR?

**Make sure:**
- Phone camera is clean
- Good lighting
- Hold phone 6-8 inches from screen
- QR code is fully visible on screen

**Try:**
- Move phone closer/farther
- Adjust angle
- Tap "Link a Device" again

---

### "QR Code Expired"?

**Normal** - QR codes expire after 60 seconds

**Solution:**
- Refresh the WhatsApp Web page
- New QR code appears
- Scan quickly

---

### Still Shows Error After Scan?

**Check session:**
1. Click **"Check Session"** button
2. Should show: **"✅ Session found! (X files)"**
3. If shows warning, re-scan QR code

**Try again:**
1. Close WhatsApp Web tab
2. Click "Test WhatsApp" again
3. Wait up to 30 seconds
