## Getting Started

This is your personal AI employee workspace. Here's how it works:

### Folder Overview
| Folder | Purpose |
|--------|---------|
| `/Inbox` | Drop new files/tasks here |
| `/Needs_Action` | Items waiting for decision |
| `/Plans` | Task plans and roadmaps |
| `/Done` | Completed work |
| `/Logs` | Activity records |
| `/scripts` | Automation code |

### How to Use
1. Drop files into `/Inbox` for processing
2. AI will categorize and move to `/Needs_Action`
3. Review and prioritize tasks
4. Complete work and move to `/Done`

### Quick Commands
```bash
# Start inbox watcher
python scripts/watcher_inbox-monitor_v1.0.py

# Check vault health
python scripts/util_file-mover_v0.5.py --health
```
